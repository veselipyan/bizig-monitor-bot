# Bizig / OpenClaw Workspace

Этот репозиторий содержит правила, архитектуру и состояние AI‑ассистента **Bizig**, работающего внутри OpenClaw.

## Структура

- `cursor_temp/` — глобальные правила и состояние Bizig
  - `BIZIK_RULES.md` — правила поведения и workflow
  - `Bizig2Dev.md` — трекер задач (единый источник правды)
  - `architecture.md` — архитектура workspace и компонентов
  - `DEVELOPMENT_HISTORY.md` — история развития
- `projects/` — каталоги конкретных проектов (боты, сервисы, скрипты)
- `repositories/` — клоны git‑репозиториев проектов
- `HEARTBEAT.md` — задачи автопилота Bizig (что нужно периодически проверять/доделывать)

## GitHub репозиторий

## Bizig Telegram Bot (план)

Отдельный проект на **Python + aiogram v3**, код предполагается держать в репозитории
`bizig-telegram-bot` (отдельный GitHub‑репозиторий) и локально в `repositories/bizig-telegram-bot/`.

Бот будет:
- показывать статус задач и проектов из `Bizig2Dev.md`;
- показывать правила и архитектуру;
- управлять через reply‑клавиатуру и inline‑кнопки;
- в будущем — принимать голосовые OGG и раскладывать текст по задачам/правилам.
